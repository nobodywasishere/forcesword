About Forcesword

Forcesword is an alternative frontend for SourceForge and other Apache Allura websites. Built around the Allura REST API with Flask and Bootstrap. The source code is available here, licensed under the GPLv3 license.

To use, simply replace sourceforge.net with forcesword.net for any Sourceforge project or user endpoint. For example, https://sourceforge.net/projects/unicon/ would become https://forcesword.net/projects/unicon/
